Solve math problems instantly with our free online Fraction Calculator. Add, subtract, multiply, divide, and simplify fractions or mixed numbers with ease.

[https://www.calculator.io/fraction-calculator/](https://www.calculator.io/fraction-calculator/)

== Usage ==

Add the Fraction Calculator shortcode to your page, post or sidebar:

`[calcio_fraction_calculator]`

Add the following code to your template where you would like the Fraction Calculator to appear:

`<?php display_calcio_fraction_calculator(); ?>`

== Installation ==

1. Upload the Fraction Calculator /calcio_fraction_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Fraction Calculator plugin through the "Plugins" menu in WordPress.
